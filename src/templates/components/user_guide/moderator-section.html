Фильтры «Дата создания с» и «Дата создания по» позволяют отображать только объекты, созданные в заданный промежуток времени. Пакетная верификация объектов

Существует возможность пакетной верификации объектов. Выбор объектов, для изменения статуса, может осуществляться двумя способами: при помощи фильтров, описанных ранее или при помощи установки флажков в первом столбце строки, интересующего объекта (см. рисунок 3.18)

Рисунок 3.18 – Внешний вид флажков для выбора объектов

Рассмотрим первый способ подробнее. После выбора значения, отличного от значения по умолчанию любого из фильтров, становятся доступными кнопки «Принять все» и «Отклонить все», нажатие на которые запустит выполнение действия, по изменению статуса всех объектов, удовлетворяющих фильтру, а не только представленных на странице. При нажатии на любую из кнопок появится модальное окно, с просьбой подтверждения пакетной верификации объектов (см. рисунок 3.19) и, при нажатии на кнопку «Да» произойдёт изменение статуса у всех моделей, соответствующих фильтру.

Рисунок 3.19 – Внешний вид модального окна подтверждения пакетной верификации

При просмотре расширенной информации о статусе объекта будет отображаться сообщение, гласящее, что статус объекта был изменён в ходе пакетной верификации (см рисунок 3.20)

Рисунок 3.20 – Информация о пакетной верификации объектов в их статусе

Рассмотрим второй способ выбора объектов для пакетной верификации. При его использовании для выбора объектов необходимо устанавливать описанные ранее флажки в строках таблицы, соответствующих интересующим моделям. После установки хотя бы одного флажка название кнопок действий внизу страницы меняется на «Принять выбранные» и «Отклонить выбранные». Теперь данные кнопки будут осуществлять действия с объектами вне зависимости от того, какие фильтры выбраны до тех пор, пока все флажки не будут сняты.

Во всех остальных аспектах верификация посредством выбора конкретных объектов установкой флажков в соответствующих строках таблицы ничем не отличается описанного ранее процесса.
