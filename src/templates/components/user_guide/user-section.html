Аутентификация в системе

Аутентификация в системе производится путём ввода данных учётной записи пользователя в поле ввода на стартовой странице. Верхнее первое вводится логин пользователя, во второе вводится токен аутентификации, выданный пользователю администратором. Страница авторизации представлена на рисунке 3.2

Рисунок 3.2 – Внешний вид формы аутентификации в системе

Поиск в системе

После успешной аутентификации пользователь увидит стартовую страницу, которая позволит осуществлять поиск информации в системе (см. рисунок 3.3).

Рисунок 3.3 – Внешний вид стартовой страницы

Поисковый запрос вводится в соответствующее поле и, после нажатия кнопки «Найти» осуществляется поиск среди доступных пользователю объектов. Внешний вид страницы с результатами поиска представлен на рисунке 3.4

Рисунок 3.4 – Внешний вид страницы с результатами поиска

Загрузка данных путём заполнения веб–формы

Второй доступной страницей является «Добавление моделей» (см. рисунок 3.5).

Рисунок 3.5 – Внешний вид страницы «Добавление моделей»

Для добавления данных в систему посредством заполнения веб–форм необходимо выбрать интересующую модель, и, путём нажатия на её название перейти на страницу с автогенерируемой формой (см. рисунок 3.6)

Рисунок 3.6 – Внешний вид страницы с автогенерируемой формой

После заполнения формы данными и нажатия на кнопку «Создать объект», при успешном создании объекта выбранной модели значения в полях формы очистятся, а в верхнем правом углу появится уведомление том, что объект выбранной модели создан успешно (см рисунок 3.7).

Рисунок 3.7 – Внешний вид результата успешного создания объекта

Загрузка данных из файла

Также пользователь может загружать несколько моделей одновременно путём загрузки данных их файла. Для этого на странице «Добавление объектов» необходимо скачать файл, содержащий шаблон заполнения для конкретной модели путём нажатия на кнопку, выделенную на рисунке 3.8

Рисунок 3.8 – Внешний вид кнопки загрузки файла, содержащего шаблон для множественной загрузки

Содержимое файла представлено на рисунке 2.4. После его заполнения, для загрузки внесённой в файл информации в систему, необходимо нажать кнопку «Загрузить объекты из файла», после чего, в появившемся модальном окне выбрать заполненный файл и нажать на кнопку «Загрузить» (см. рисунок 3.9)

Рисунок 3.9 – Внешний вид модального окна загрузки файла с объектами
